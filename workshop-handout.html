Deploying a Secure Autonomous Agent

ODSC 2026  ·  Hands-On Workshop  ·  Please download/install before the session

Wi-Fi tip: Download installers and save account pages as PDFs before you arrive. The workshop notebooks run locally — you only need internet access for initial API calls.

1   Software to Install

Docker Desktop (required)

Runs both agents in isolated containers.

  • Mac (Apple Silicon): Docker Desktop dmg
  • Mac (Intel): Docker Desktop dmg
  • Windows: Docker Desktop exe
  • Linux: Docker Engine + Compose plugin

Download: docker.com/products/docker-desktop

Python 3.10+ (required)

Most attendees will already have this.

  • Mac: use Homebrew — brew install python
  • Windows / Linux: python.org/downloads

Verify: python --version

Git (required)

Download: git-scm.com/downloads

3   Telegram Bot Setup (5 min — do this before arriving)

1
Open Telegram and search for @BotFather. Send /newbot.
2
Choose a display name (e.g. Compliance Alert Bot) and a username ending in bot.
3
BotFather replies with your HTTP API token — save it.
4
Send your bot any message (e.g. "hello"), then open:
https://api.telegram.org/bot<YOUR_TOKEN>/getUpdates
Find "chat":{"id": 123456789} — that number is your chat ID.

4   Get the Repo Ready

1
Clone: git clone https://github.com/saskinosie/odsc-2026-autonomous-compliance-agent.git
2
Virtual env: python -m venv .venv && source .venv/bin/activate  (Windows: .venv\Scripts\activate)
3
Dependencies: pip install -r requirements.txt
4
Docker pull (saves time on-site): docker compose -f custom-claw/docker-compose.yml pull

OpenClawAgent framework running in Docker. Same logic as Custom Claw but with scheduling, a web dashboard, and 50+ built-in skills.
